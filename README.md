# 🧩 Block Blast PC Clone
A dedicated PC clone of the popular **Block Blast** game originally found on Android devices. Rebuilt from the ground up to provide the same addictive puzzle experience on your desktop.

## 🚀 Key Features
- **Standalone Execution:** No Python installation required. Just download the ZIP and play.
- **Enhanced Audio:** High-quality sound effects for block placement and row clears.
- **Optimized Performance:** Clean UI and smooth gameplay, tailored for PC environments.

## 🆙 Update (v1.5)
- **New Visuals:** Added "Milk-Yellow" line-clear highlights with smooth fade-out effects.
- **Praise System:** Added animated `GOOD!`, `COMBO!`, and `EXCELLENT!` text.
- **Improved Logic:** Added a 2-second delay before the Game Over screen appears.

## 📥 How to Play
1. Look at the file list above and click on **`Block blast v1.5.zip`**.
2. On the next page, click the **Download** button (on the right side) to get the file.
3. Extract the contents of the ZIP file into a folder on your PC.
4. Run **`Block_Blast_Game.exe`** to start playing!

## 🛠 Tech Stack
- **Language:** Python 3.11
- **Framework:** Pygame
